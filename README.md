# TP-noté
Ecrire une fonction qui : 
  - demande une IP et un Masque à l'utilisateur.rice
  - affiche l'IP du réseau
  - affiche l'IP de diffusion
  - donne l'IP du routeur (la dernière dispo)
  - donne l'IP du serveur DHCP (première dispo)
  - donne la plage adressable : de ... à ...
  - affiche le nombre de places dans le réseau : 2^(32-masque) - 4
